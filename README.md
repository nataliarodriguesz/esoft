# 📚 Repositório Acadêmico – Engenharia de Software

Este repositório reúne atividades, exercícios e projetos desenvolvidos durante a graduação, com foco no aprendizado prático de programação, lógica e sistemas digitais.

---

## 📁 Estrutura do Repositório

O repositório está organizado da seguinte forma:

### 🤖 AEP – Chatbot de Saúde
Projeto desenvolvido como atividade prática, com foco em lógica de programação e simulação de um sistema de atendimento básico na área da saúde.

---

### 🧠 Algoritmos
Exercícios e atividades da disciplina de Algoritmos, incluindo:
- Estruturas condicionais
- Laços de repetição
- Vetores e funções
- Resolução de problemas lógicos

---

### 💻 Dev-C++
Projetos desenvolvidos utilizando a IDE Dev-C++, geralmente em linguagem C/C++, com foco em:
- Estruturas básicas da linguagem
- Manipulação de dados
- Exercícios práticos da disciplina

---

### 🔌 Logisim
Atividades de circuitos digitais desenvolvidas na ferramenta Logisim, abordando:
- Portas lógicas
- Circuitos combinacionais
- Circuitos sequenciais
- Simulação de lógica digital

---

## 🎯 Objetivo

Este repositório tem como objetivo centralizar os conteúdos desenvolvidos ao longo do curso, facilitando o acompanhamento da evolução acadêmica e servindo como base para prática contínua de programação e lógica computacional.

---

## 👩‍💻 Autoria

Repositório acadêmico pessoal utilizado para organização de atividades da graduação.
